# 公安派出所报警管理系统－－日常笔录管理版

主要功能如下：  
一、报警登记：报警案件登记表、报警案件查询  
二、值班登记表：值班登记表   
三、案件管理：行政治安查询、治安查询、刑事案件查询、案件管理、行政治安登记表、  刑事登记簿、治安登记表
